What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications.

Basic HTML Document

                
                    <!DOCTYPE html>
                    <html>
                    <head>
                    <title>Page Title</title>
                    </head>
                    <body>

                    <h1>This is a Heading</h1>
                    <p>This is a paragraph.</p>

                    </body>
                    </html>
